Bicycle Rush (BIR) is a Web3 application combining GameFi and SocialFi elements that allows users to Move & Earn (cycling and earn)

Users need to equip themselves with NFT DAPP. By cycling, outdoors, users will earn in-game tokens.

Bicycle Rush (BIR) aims to motivate millions of people towards healthier lifestyles, fight climate change and connect the public with Web 3.0, while addressing its Social-Fi aspect to build a lasting foundation that drives user-generated Web 3.0 content.

Bicycle Rush (BIR) users will be able to earn in-game tokens by moving which requires a minimum of one NFT DAPP
